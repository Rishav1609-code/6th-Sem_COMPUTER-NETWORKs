# 📚 Computer Networks - 6th Semester

Welcome to the **Computer Networks** repository! This project serves as a centralized, structured hub for third-year engineering study materials, detailed notes, and frequently asked questions (FAQs) dedicated to the Computer Networks curriculum.

---

## 📁 Repository Structure

The course material is carefully organized into distinct units. Each directory contains modular `.md` files breaking down specific topics, alongside dedicated FAQ documents for quick revision and conceptual clarity.

### 📖 UNIT-1
Focuses on the foundational principles and introductory concepts of computer networking.
* **Study Modules:** 9 detailed topic files (`UNIT-1_1.md` to `UNIT-1_9.md`).
* **Resources:** `FAQs.md` – A collection of common questions and core takeaways for exam preparation.

### 📖 UNIT-2
Dives deeper into advanced network protocols, layers, and architectures.
* **Study Modules:** 14 comprehensive topic files (`UNIT-2_1.md` to `UNIT-2_14.md`).
* **Resources:** 
  * `FAQs.md` – Standard question bank.
  * `FAQs_MAX.md` – An extended, deep-dive question bank for thorough revision.
